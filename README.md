# AGNT v1.56.6 — Persistent Sessions & Metric Sync

Built incrementally from v1.56.5.

## Changes
- Sessions remain active until End Session is pressed.
- Save & Next loads the next queued contact inside the same session.
- Prospector outcomes write Calls and Connects to the same daily record used by Home.
- End Session opens a review showing calls, connects, warm/hot leads and appointments booked.
- Duplicate metric application remains protected by interaction IDs.

## Firebase
No configuration, rules or path changes required.
