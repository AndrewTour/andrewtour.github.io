# AGNT V69 — Seller Pipeline Foundation

Built incrementally from the confirmed stable AGNT V68 release.

## Changes
- Built the Prospector Pipeline tab with Now, 1–3 months, 6–12 months and 12 months+ sections.
- Reused the existing contact detail and interaction-history interface.
- Added a selling-timeframe field to contacts and session outcomes.
- Connected new Listing Appointments to matching contacts and the seller pipeline.
- Reused existing temperature and follow-up records without duplicate tasks or collections.
- Added contact-history entries when selling timeframe changes.
- Added compact pipeline filtering and sorting.

## Protected systems unchanged
Firebase configuration, authentication, Firestore paths and rules, UID separation, daily accountability calculations, leaderboard logic, local cache keys and deployment architecture remain unchanged.

No Firebase configuration or rules changes are required.
