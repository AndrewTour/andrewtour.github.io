# AGNT V90 — Booking UI Cleanup

Incremental update built from V89.

## Changed
- Removed the standalone appointment follow-up reminder box from the booking page.
- Added the same live reminder text beneath the Past Appointments summary as a bold yellow metric line.
- Kept the normal booking form free of editing labels and controls.
- Preserved the isolated appointment editor and its existing save behaviour.

## Unchanged
Firebase, authentication, Firestore paths and rules, UID separation, local cache, sync, appointment data, navigation, scoring and unrelated UI.
