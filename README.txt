DAILY ACCOUNTABILITY PWA

Upload every file and the icons folder to the root of your GitHub Pages repository.
Do not upload only the ZIP.

Files required:
- index.html
- manifest.json
- service-worker.js
- icons/icon-192.png
- icons/icon-512.png

After uploading:
1. Commit the changes in GitHub.
2. Open your GitHub Pages URL in Safari.
3. Refresh the page once.
4. Close the Home Screen app completely and reopen it.

This version includes:
- 12-hour time and live call recommendations
- Calls, connects and data accountability targets
- An ongoing knocking timer that does not stop at the daily target
- A 240-minute minimum weekly knocking target
- Automatic carry-forward between Monday, Tuesday, Thursday and Friday
- Extra knocking reduces the next scheduled day's requirement
- Missed knocking increases the next scheduled day's requirement
- Weekly completion scoring, history and yearly calendar
- Haptic feedback, offline storage and target alarm
